# Framework Image

Please save the framework image as `framework-full.jpg` in this directory.

The image should be:
- Optimized for web (recommended max width: 1920px)
- Compressed to reduce file size for faster loading
- Format: JPG or WebP for better compression

The modal supports:
- Click to zoom (2x)
- ESC key to close
- Click outside to close
- Mobile responsive
